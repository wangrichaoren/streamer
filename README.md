# **_Streamer_**

---

### Platforms

* Linux (Ubuntu20.04, x64, gcc-9.4.0)

---

### Dependencies

* CMake >3.8 (local=3.8)
* Qt >5.15.x  (local=3.15.2)
* Opencv >3.4.x  (local=3.4.15)
* PCL >1.8.x  (local=1.9.1)

---

### Building

* git clone git@github.com:wangrichaoren/streamer.git
* cd streamer
* mkdir build && cd build
* cmake ..
* make -j9 && make install
